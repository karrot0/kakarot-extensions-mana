## Adding to Mana

1. Open the Mana app and navigate to **Discover**
2. Click the package button
3. Tap **Plus**
4. Paste the URL:
   ```
   https://karrot.github.io/kakarot-extensions-mana/main
   ```

## Development

### Prerequisites

- [Bun](https://bun.sh) v1.0+
- Node.js 18+

### Setup

```bash
bun install
```

### Creating a New Source

1. Copy `src/Template/` to `src/<YourSourceName>/`
2. Start developing
